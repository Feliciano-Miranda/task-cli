# Task Tracker CLI

CLI simples para gerenciamento de tarefas usando Node.js e JSON.

## Funcionalidades

- Adicionar tarefas
- Atualizar tarefas
- Remover tarefas
- Marcar tarefas
- Listar tarefas

## Instalação

```bash
git clone <repositorio>
cd task-cli

Uso
Adicionar tarefa
node task-cli.js add "Minha tarefa"
Atualizar
node task-cli.js update 1 "Nova tarefa"

Remover
node task-cli.js delete 1
Marcar em andamento
node task-cli.js mark-in-progress 1
Marcar como concluída
node task-cli.js mark-done 1
Listar todas
node task-cli.js list

Filtrar por status
node task-cli.js list done
node task-cli.js list todo
node task-cli.js list in-progress
Tecnologias
Node.js
File System (fs)
JSON
```
